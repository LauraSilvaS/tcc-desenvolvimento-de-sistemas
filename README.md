# Bares & Comércio Local
Projeto desenvolvido como conclusão de curso para o curso Técnico em Desenvolvimento de Sistemas.

## Status do Projeto
Em desenvolvimento...

## Objetivo do Software
Realizar o cadastro, atualização, exclusão e listagem de itens de um comércio local.

## Funcionalidades do Sistema
- Listagem de boletos, clientes, fornecedores, vendas e produtos
- Cadastro de boletos, clientes, fornecedores, vendas e produtos
- Edição de boletos, clientes, fornecedores, vendas e produtos
- Exclusão de boletos, clientes, fornecedores, vendas e produtos

## Tecnologias Utilizadas
- Java
- MySQL
- CSS
- HTML
- JavaScript
- SpringBoot
